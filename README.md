# PDF Генератор

Консольное приложение на Python для генерации PDF-документов из CSV, JSON или Google Sheets по HTML-шаблонам.

Проект подходит для счетов, чеков, заказов, прайс-листов, каталогов и других однотипных документов, где данные хранятся в таблице, а внешний вид задается HTML-шаблоном.

## Возможности

- Генерация PDF из CSV и JSON файлов.
- Загрузка данных напрямую из Google Sheets.
- Выбор конкретного документа по `invoice_id`.
- HTML-шаблоны с плейсхолдерами вида `{customer_name}`.
- Поддержка кириллицы через системные или локальные шрифты.
- Автоматическое сохранение PDF в папку `output/`.
- Автоматическое открытие готового PDF после генерации.

## Установка

Склонируйте репозиторий:

```bash
git clone <repository-url>
cd PDF-генератор
```

Создайте виртуальное окружение:

```bash
python -m venv venv
```

Активируйте виртуальное окружение.

Windows PowerShell:

```powershell
venv\Scripts\Activate.ps1
```

Linux / macOS:

```bash
source venv/bin/activate
```

Установите зависимости:

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python pdf_generator.py
```

После запуска приложение предложит:

1. Выбрать HTML-шаблон.
2. Выбрать источник данных: локальный CSV/JSON или Google Sheets.
3. Выбрать файл данных или указать ссылку на Google таблицу.
4. Выбрать `invoice_id`.
5. Сгенерировать PDF.

Готовый файл появится в папке `output/`.

## Формат данных

Первая строка CSV или Google Sheets должна содержать названия колонок. Для JSON ожидается массив объектов.

Пример CSV:

```csv
invoice_id,customer_name,date,total,status
INV-001,Иван Петров,2024-01-15,15000.00,Оплачен
INV-002,Мария Сидорова,2024-01-16,23000.50,В обработке
```

Пример JSON:

```json
[
  {
    "invoice_id": "INV-001",
    "customer_name": "Иван Петров",
    "date": "2024-01-15",
    "total": 15000,
    "status": "Оплачен"
  }
]
```

В данных обязательно должно быть поле с идентификатором документа, например `invoice_id`.

## HTML-шаблоны

Шаблоны находятся в папке `templates/`. Внутри HTML можно использовать плейсхолдеры, которые совпадают с названиями колонок или ключей в данных.

Пример:

```html
<h1>Счет {invoice_id}</h1>
<p>Клиент: {customer_name}</p>
<p>Дата: {date}</p>
<p>Сумма: {total}</p>
<p>Статус: {status}</p>
```

Во время генерации значения из выбранной записи подставляются в шаблон, после чего WeasyPrint создает PDF.

## Google Sheets

Чтобы использовать Google Sheets как источник данных:

1. Создайте проект в Google Cloud.
2. Включите Google Sheets API.
3. Создайте Service Account.
4. Скачайте JSON-ключ.
5. Положите ключ в корень проекта с именем `credentials.json`.
6. Откройте Google таблицу и добавьте email service account в доступ с правом просмотра.
7. Запустите приложение и выберите источник `Google Sheets`.

Файл `credentials.json` добавлен в `.gitignore`, поэтому он не должен попадать в Git.

## Пример использования

Для генерации счета:

1. Добавьте данные в `data/invoices.csv` или Google Sheets.
2. Подготовьте шаблон `templates/invoice_simple.html`.
3. Запустите `python pdf_generator.py`.
4. Выберите нужный источник данных и `invoice_id`.
5. Получите PDF в папке `output/`.
